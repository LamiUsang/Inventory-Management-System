## Description

## Features
- [ ] Add products (standard, perishable, digital)
- [ ] Sell and restock products
- [ ] Search by name or category
- [ ] Low stock alerts
- [ ] Stock value report by category
- [ ] Transaction history log
- [ ] Data persistence (JSON)
- [ ] Logging to console and file
- [ ] Custom exceptions
- [ ] Test suite (10+ tests)
